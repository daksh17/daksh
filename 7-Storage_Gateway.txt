Storage Gateway
	Bridge between AWS Storage Infrastructure and On-Prem Env
	Virtual Machine on Data Center Host or On-Prem
	Supports ESXi and Hyper-V, Physical Machine and EC2 

Types
	1. File Gateway (NFS) for Flat Files: Data gets stored into S3
	
	2. Volume Gateway (iSCSI)	for Block level files or Flat Files: data gets stored into EBS volumes
		a. Stored Volume (All Data on EBS Snapshot and Local Data Center) 1GB - 16 TB
		b. Cached Volume (Frequestly Used data on Local Data Center, rest on AWS EBS) upto 32 TB
	
	3. Tape Gateway (Data gets stored in glaciers)
		Virtual Tape Gateway to Archive into Glacier
